## Project Overview

This project is an **interactive Power BI dashboard** designed to analyze **blinkit sales data** and present insights in a clean, business-friendly format. The dashboard focuses on understanding sales performance across **outlets, items, and time**, using intuitive navigation, filters, and KPI-driven visuals inspired by real-world product dashboards.

The project is suitable for **data analyst portfolios**, demonstrating skills in **Power BI, data modeling, DAX, and dashboard design**.

---

## Objectives

- Analyze overall sales performance
- Compare outlet types, sizes, and locations
- Understand item-level contribution to revenue
- Provide an interactive, user-friendly reporting experience
